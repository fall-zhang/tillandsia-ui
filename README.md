# tillandsia

> tillandsia 空气凤梨，没有土壤，在空气中，加入适当的水即可存活
> 希望该 UI 也同空气凤梨一样，只需要很少的配置即可运行

## 开始使用

你只需要拥有 tailwindcss + react 的运行环境，然后复制粘贴即可使用该 ui 库
